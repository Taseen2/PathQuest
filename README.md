# 🪔 The Lost Lamp of Varanasi

> *A choice-driven narrative game set on the night of Dev Deepawali — where a sacred flame, a stolen lamp, and the oldest city in the world are waiting for you.*

---

## 📖 About

**The Lost Lamp of Varanasi** is an interactive story built with pure HTML, CSS, and vanilla JavaScript. No frameworks. No libraries. Just a city, a missing lamp, and a rising moon.

Three paths. Nine unique endings. One city that earns its mysteries.

---

## ✨ Features

- **3 main story paths** — the Ganga ghats, the market lanes, the temple tower
- **9 distinct endings** — Happy, Mysterious, or Bad, each with real consequences
- **Live interactive riddle** — one path requires you to type your answer to unlock the ending
- **Atmospheric UI** — flickering diya animations, animated river reflection, procedural starfield, all in CSS
- **No dependencies** — single HTML file, runs anywhere
- **Branching narrative with memory** — a back button lets you retrace your steps
